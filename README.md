## Questions and Justification:
Question 1: What is the average lead time (time from when the alert is sent and when the event actually begins) to prepare for Extreme vs. Severe events, and which specific event type provides the shortest window for public response? The relevant columns are ALERT_INSTRUCTION, EVENT_URGENCY, EVENT_CATEGORY, and NWS_ALERT_ID. This question is important because in a crisis, brevity can be life-saving. If immediate alerts contain significantly more text than future alerts, it might suggest that the NWS is overwhelming the public with information exactly when they would have the least amount of time to read it.
